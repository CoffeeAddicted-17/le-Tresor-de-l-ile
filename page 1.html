Comment jouer ?

Vous allez devoir résoudre un maximum d’énigmes dans le temps imparti.

Chaque énigme, une fois résolue, vous donnera une lettre que vous devrez replacer à sa place sur la page précédente.

Si vous fonctionnez en plusieurs équipes, chaque énigme résolue vous rapportera des points...

À l’issue du temps imparti, plusieurs cas de figure :

  • 1 : Vous avez réussi toutes les énigmes. Rendez-vous au lieu où se cache le trésor pour l’énigme finale.
  • 2 : Vous n’avez pas réussi toutes les énigmes, mais vous avez deviné le lieu où se cache le trésor. Allez-y pour l’énigme finale.
  • 3 : Vous n’avez pas réussi toutes les énigmes et vous ne devinez pas où se trouve le trésor. Ouvrez l’enveloppe...
